# Проект для фильтрации банковских операций и данных и тестирования функций 

## Описание:

Проект представляет домашнюю работу по взаимодействию локального репозитория и GitHub 
и созданию и редактированию веток, а также для тестирования функций. 

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/vseokolo/hw_10_1_naumova
```
2. Установите зависимости:
```
pip install -r requirements.txt
```


## Документация:
Содержимое пакета src:
- Модуль masks.py маскирует номер карты и банковского аккаунта.
- Модуль processing.py фильтрует операции по статусу и дате. 
- Модуль widget.py маскирует номер карты, также возвращая ее имя; форматирует дату.
Содержимое пакета tests:
- Модуль test_mask_account.py тестирует функцию get_mask_account.
- Модуль test_mask_card_number тестирует функцию get_mask_card_number.
- Модуль widget_mask_account_card.py тестирует функцию mask_account_card.
- Модуль widget_get_date.py тестирует функцию get_date.
- Модуль test_processing_state.py тестирует функцию filter_by_state.
- Модуль  test_processing_date тестирует функцию sort_by_date.


#### Пример для пакета с функциями:
На вход подается номер карты. Система проверяет, содержится ли там 16 цифр.
Система максирует номер карты с помощью функции get_mask_card_number и возвращает зашифрованный номер.

#### Пример для тестов:
Чтобы определить процент покрытия функций тестами, найдите папку htmlcov и откройте файл index.html.
