# Проект для фильтрации банковских операций и данных

## Описание:

Проект представляет домашнюю работу по взаимодействию локального репозитория и GitHub 
и созданию и редактированию веток. 

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/vseokolo/hw_10_1_naumova
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Использование:
```
1. В папке src найдите модули с функциями, описанные ниже. 
```
```
2. Тестируйте функции, вводя нужные данные.
```

## Документация:
Содержимое пакета src:
- Модуль masks.py маскирует номер карты и банковского аккаунта.
- Модуль processing.py фильтрует операции по статусу и дате. 
- Модуль widget.py маскирует номер карты, также возвращая ее имя; форматирует дату.

#### Пример:
На вход подается номер карты. Система проверяет, содержится ли там 16 цифр.
Система максирует номер карты с помощью функции get_mask_card_number и возвращает зашифрованный номер.
